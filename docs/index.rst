Development
-----------

You can build the development environment locally by running these commands in the terminal:

.. code-block:: bash

    $ git clone https://github.com/ImJuanan/flask-milligram.git
    $ cd flask-milligram
    $ pipenv install
    $ pipenv run pip install -e ".[dev]"
    $ tox
